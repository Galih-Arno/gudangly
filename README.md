#aplikasi manejemen inventaris

##TUGAS

#FRONTEND -> berkaitan dengan tampilannya
- perbaiki tampilannya semenarik mungkin, base code yang saya kasih menggunakan HTML, CSS Bootstrap, Javascript sederhana
- tambahkan fitur untuk menambahkan, menghapus, dan mengubah data inventaris
- tambahkan fitur untuk mencari data inventaris
- tambahkan fitur untuk menampilkan data inventaris secara keseluruhan
- tambahkan fitur untuk menampilkan data inventaris berdasarkan kategori
- tambahkan fitur untuk menampilkan data inventaris berdasarkan harga
- tambahkan fitur untuk menampilkan data inventaris berdasarkan stok

#BACKEND -> berkaitan dengan database
- buat database untuk menyimpan data inventaris di pc kalian, disini saya bantu menggunakan mysql (XAMPP)
- siapkan databasenya dahulu, cek file app.py untuk konfigurasi koneksi python ke mysql
- lihat file create_inventory_db.sql untuk format tabel pada database

##WARNING
- pastikan kalian sudah menginstall XAMPP di pc kalian
- pastikan kalian sudah menginstall python di pc kalian
- pastikan kalian sudah menginstall flask di pc kalian
- pastikan kalian sudah menginstall flask to mysql di proyek ini
